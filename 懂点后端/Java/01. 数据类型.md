# 数据类型

## 基本数据类型

| 分类   | 数据类型       | 内存占用 | 数据范围         |
| ------ | -------------- | -------- | ---------------- |
| 整数   | byte           | 1B       | -128 ～ 127      |
| 整数   | short          | 2B       | 约 ±3万          |
| 整数   | int（默认）    | 4B       | 约 ±21亿（10位） |
| 整数   | long           | 8B       | 19位             |
| 浮点数 | float          | 4B       | —                |
| 浮点数 | double（默认） | 8B       | —                |
| 字符型 | char           | 2B       | —                |
| 布尔型 | boolean        | 1B       | true / false     |

- `char` 只能使用单引号，有且仅有一个字符
- 整数字面量默认是 `int`，加 `L/l` 后缀变为 `long`
- 浮点数字面量默认是 `double`，加 `F/f` 后缀变为 `float`

```java
byte b = 127;
short s = 32767;
int i = 2147483647;
long l = 9223372036854775807L;
float f = 17.17F;
double d = 17.17;
char c = 'a';
```

## 类型转换

### 自动类型转换

> 类型范围小的变量，可以**直接赋值**给类型范围大的变量 —— "大鱼吃小鱼，小鱼吃虾米"

在表达式中，小范围类型的变量，会自动转换成表达式中较大范围的类型，再参与运算。

`byte`、`short`、`char` → `int` → `long` → `float` → `double`

- 在表达式中，`byte`、`short`、`char` 是**直接转换成 **`int`** 类型**参与运算的
- 表达式的最终结果类型由表达式中的**最高类型决定**

### 强制类型转换

> **类型范围大**的数据或变量，直接赋值给**类型范围小**的变量，会报错

语法：

```java
数据类型 变量 = (数据类型) 变量/数据;
```

- **非必要不要使用**，强制类型转换可能造成**数据溢出（丢失）**
- 浮点型强转成整型，**直接丢掉小数部分，保留整数部分**

## 运算符

- Java 中两个整数相除**结果还是整数**
- 整数相除想得到小数：`1.0 * i / j` 或 `(double) i / j`
- Java 中没有 `===`，比较引用数据类型需使用对应的 `.equals()` 方法

## 引用数据类型

### 数组

Java 数组是**强类型、定长**的，元素必须同类型，长度创建后不可变，越界访问抛出 `ArrayIndexOutOfBoundsException`。

### String

String 对象的内容不可改变，被称为**不可变字符串对象**；每次修改都会产生新对象。

- 以双引号 `"..."` 方式创建的字符串存储在**字符串常量池**，相同内容只存一份
- 通过 `new` 方式创建的字符串，每次都会在堆内存中产生一个新对象
